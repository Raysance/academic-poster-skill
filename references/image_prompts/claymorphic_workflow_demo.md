# Claymorphic Horizontal Workflow / Demo Prompt

Use for presentation-friendly workflow diagrams, project pipelines, robotics demos, physical-system demos, milestone timelines, and polished explanatory figures where the goal is clarity and visual appeal rather than exact quantitative plotting.

```text
A professional horizontal infographic timeline for an AI tech project workflow diagram, 3D render style.

[Layout and Flow]
Create a wide horizontal diagram with 6 sequential vertical white cards aligned side-by-side.
Each card has soft rounded corners, subtle drop shadows, and a clean white surface.
Arrange the cards from left to right as steps 1 to 6.
Place small circular number badges at the top of each card, alternating corporate blue and emerald green.
Use simple dark blue arrow connectors between cards to create a clear step-by-step pipeline.

[Visual Elements per Card]
- Card 1: A cute 3D clay-style document icon with a blue target and arrow next to it.
- Card 2: A miniature 3D robotic arm on a wooden platform picking up colored blocks.
- Card 3: A central 3D microchip icon with glowing nodes and abstract data connection lines.
- Card 4: A 3D clipboard checklist icon with checkmarks and slider controls at the bottom.
- Card 5: A 3D vertical bar chart with a prominent turquoise bar showing data growth.
- Card 6: A final 3D robotic arm model ready for a deployment demo.

[Text Placeholders]
Leave clean typography placeholders on each card for a short step title and one line of subtext.
Use generic labels unless project-specific step names are supplied.

[Aesthetics and Rendering]
Claymorphic style, soft matte 3D clay texture, minimalist clean interface design.
Use smooth lighting and a bright vibrant palette with corporate blue, emerald green, and purple accents.
Keep the composition vector-like, polished, and presentation-ready.
Place the full diagram on a solid light grey-blue background.

[Output]
Aspect ratio 16:5. High resolution. No watermark. No extra explanatory paragraphs outside the diagram.
```

## Adaptation notes

- Use this template for visual workflow/demo figures, not for exact numeric plots.
- Replace the six card icons and labels with the approved project stages when available.
- For real experiment results, generate or validate charts with Python first, then use AI only for visual packaging.
